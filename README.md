## 內容
透過APP的引導，讓車禍發生後讓駕駛人有概念處理車禍事故<br>
例如：
- 車禍車損拍照引導
- 保險專員分配與聯絡
- 事故處理流程引導
- 協助通報警方
- 車禍事故地點分享至警方或急救人員
